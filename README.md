# Практическая работа по проектированию ПО

## Тема
Диаграммы классов и объектов с использованием языка UML.

## Описание проекта
Данный проект представляет собой консольное Java-приложение для учета кораблей.
В программе реализована иерархия наследования для четырех сущностей:

* корабль;
* пароход;
* парусник;
* корвет.

Базовым классом является `Ship` - корабль. От него наследуются классы `Steamship` - пароход, `SailBoat` - парусник и `Corvette` - корвет.

Приложение позволяет:

* добавлять новые объекты кораблей;
* удалять корабль по `id`;
* выводить список всех добавленных кораблей;
* сравнивать два корабля по `id`;
* завершать работу программы.

Помимо этого, для программы построены UML-диаграммы всех классов и объектов.

## Что внутри проекта
* Папка Ship - консольное Java-приложение
* файлы с диаграммой классов и диаграммой объектов
* файлы с изображением диаграммы классов и диаграммы объектов

## Как запустить программу

Для запуска программы на компьютере должен быть установлен JDK.

1. Откройте терминал.
2. Перейдите в папку проекта, содержащую файлы с расширением `.java`.
3. Выполните компиляцию исходных файлов:
javac -encoding UTF-8 *.java
4. Запустите следующий класс программы:
java Main
